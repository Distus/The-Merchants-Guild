# The Merchant's Guild

A FoundryVTT module that procedurally generates shops, shopkeepers, and inventories for **D&D 5e (2024 ruleset)**.

Designed to make shopping encounters fast for the DM and immersive for players — without feeling like a video game store UI.

## Features

- **Procedural Shop Generation** — Select a store type and town affluence tier, and the module generates a complete shop with inventory, shopkeeper, and description
- **10 Store Types** — General Goods, Blacksmith, Armorer, Apothecary, Fletcher, Magic Shop, Tavern, Stables, Jeweler, Clothier
- **5 Affluence Tiers** — Hamlet through Metropolis, affecting inventory depth, pricing, and magic item availability
- **Party Level Gating** — Item rarity is capped by average party level so low-level parties can't buy overpowered gear
- **Shopkeeper Generator** — Unique names, personalities, physical quirks, and light stat blocks for every merchant
- **Social Interaction DCs** — Quick-reference Haggle, Intimidation, Theft, Perception, and Insight DCs per shopkeeper
- **Shopkeeper Bias** — Rare negative biases against certain races or classes for RP flavor (toggleable)
- **Dual Item Descriptions** — DM sees full mechanics; players see vague flavor text only. Discovery through RP, identification, and attunement
- **Full Transaction System** — Buy and sell with automatic currency handling and coin conversion
- **Persistent Shops** — All shops are saved by name and persist across sessions
- **Configurable** — Player visibility, buyback rates, party level detection, and bias system all adjustable in module settings

## Requirements

- Foundry VTT v11+ (verified on v12)
- D&D 5e System v3.0.0+

## Installation

1. In Foundry VTT, go to **Settings → Manage Modules → Install Module**
2. Paste the following manifest URL:
   ```
   https://github.com/Distus/The-Merchants-Guild/releases/latest/download/module.json
   ```
3. Click **Install**
4. Enable "The Merchant's Guild" in your world's module settings
